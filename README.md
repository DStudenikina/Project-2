Курсы ДПО "Компьютерная лингвистика"
Учебный проект №2

Анализ перевода авторских номинаций как маркера идиостиля переводчика (на примере романа Дж.Р.Р. Толкина “Властелин Колец”)


**Аннотация**
Проект посвящен анализу идиостиля переводчиков романа Дж.Р.Р. Толкина “Властелин Колец”. Для данного проекта основным признаком, характеризующим особенности языковой личности авторов перевода, является перевод авторских номинаций, которые моделируют художественный мир “Властелина Колец” – топонимы, имена персонажей, названия событий и др. Например, в романе “Властелин Колец” такими единицами являются Frodo Baggins, Sméagol, Bag End, Shire. 

Особенность переводов подобных номинаций заключается в том, что они не являются частью привычного лексикона и не являются единицами переводных словарей. 
Объекты реального мира имеют устоявшиеся единицы для перевода на другой язык. Авторские номинации, напротив, являются результатом творчества автора, средством передачи индивидуальных и культурных характеристик его языковой личности, концептов его родной (английской) языковой картины мира.
Переводчик же, будучи самостоятельной языковой личностью, сформированной в концептосфере иной (русской) лингвокультуры, сталкивается с задачей переноса этих уникальных единиц из чужой картины мира в свою. Он вынужден не просто подбирать соответствия, а заново концептуализировать авторские образы, создавая в принимающем языке номинации, которые смогут выполнять сходные художественные функции.
Этот процесс вторичной концептуализации неизбежно несет на себе отпечаток языковой личности переводчика — его картины мира, творческих установок и предпочтений. Таким образом, выбор стратегии перевода авторской номинации становится маркером идиостиля переводчика, а анализ этих маркеров позволяет реконструировать особенности его языковой личности.
 
Поэтому перевод таких ЛЕ единиц вызывает интерес. В аспекте данной работы именно языковая личность переводчика выступает объектом исследования. 

**Гипотеза:** выбор способа перевода авторской лексики является достаточно устойчивым и уникальным для каждого переводчика, чтобы служить основой для автоматической атрибуции перевода.


**Цель работы:**
Оценить степень уникальности идиостиля каждого переводчика на основе анализа перевода авторской лексики

**Задачи:**
Подготовить корпус параллельных контекстов (оригинал + 3 перевода)
Обучить модель узнавать переводчика на основе перевода авторских номинаций (на корпусе А)
Проверить, узнает ли модель переводчика (на корпусе В с другим набором номинаций и на корпусе С с замаскированными номинациями)

**Этапы:**
Подготовка корпуса параллельных переводов (оригинал + переводы)
 Аннотировать параллельный корпус (отметить, какая номинация используется в данном фрагменте)
Разделить на корпусы (для корпуса B фрагменты с номинациями, которые не встречались в корпусе А. Для корпуса С использовать новые фрагменты текста, но со скрытыми номинациями в переводах)
Обучить модель на корпусе А, используя таблицу переводов номинаций*
Проверить способность предсказания переводчика по корпусу В
Проверить способность предсказания переводчика по корпусу С

* *Готовая таблица номинаций по всем переводчикам с сайта https://kniga2001.narod.ru/tolkien/01pere/02.htm*

**Понятия:**
Номинация —  процесс и результат именования, то есть присвоения имени обозначаемому объекту.
Идиостиль – совокупность характерных языковых и текстовых особенностей, которые отличают носителя языка
Языковая личность – совокупность способностей и характеристик человека, которые определяют создание и восприятие им речевых произведений

**1. Подготовка корпуса**
Параллельный корпус был создан методом автоматического выравнивания предложений на основе многоязычных эмбеддингов. 
Получено 7254 строки с параллельными фрагментами

**2. Аннотирование параллельного корпуса**
Поиск имен во фрагментах по таблице nom.xlsx. Добавление столбцов с номинациями. 

**3. Разделение на корпусы вручную**
корпус А – самое большее количество фрагментов с номинациями
корпус В – фрагменты с другим набором номинаций
корпус С – фрагменты со скрытыми номинациями

**Проблемы исследования:**

Дисбаланс классов.
Признаки сформулированы только для номинаций.
Отсутствует эксперимент по оценке стилистических различий (без фокуса на именах)
Проблемы с формированием корпуса.
Необходимость в доработке таблицы номинаций.

**Вывод:**
Проведенное исследование показало, что стратегии перевода авторской лексики являются устойчивым маркером идиостиля переводчика. 

НО: без имен модель не находит других различий и просто относит все тексты к наиболее частотному переводчику. Это не означает, что стилистических различий нет — это означает, что для их обнаружения нужны другие методы и другие признаки. 
Исследование демонстрирует, что перевод номинаций — сильный и самодостаточный маркер, но вопрос о целостном идиостиле остается открытым для дальнейших исследований.
